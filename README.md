# CRUD_API_Node_js

### Setup

#### 1. Clone my repo

```console
$ git clone https://github.com/Thunderbolt9/CRUD_API_Node_js.git
```

#### 2. Install the dependencies

```console
$ cd CRUD_API_Node_js
$ npm install
```

#### 3. Create uploads folder
```
mkdir uploads
```

#### 4. Run project
```console
$ npm start
```
