### Setup

#### 1. Clone my repo

```console
$ git clone https://github.com/Rakshit-gpt/Nodeja_assginment
```

#### 2. Install the dependencies

```console
$ cd Node_Js_Assignment
$ npm install
```

#### 3. Create uploads folder
```
mkdir uploads
```

#### 4. Run project
```console
$ npm start
```
